Tips & Shortcuts

Share your view. Click Share in the toolbar to generate a link that captures your current variable, timescale, reference period, and map position. Anyone opening the link will see exactly what you see.
Save a view. Click Share to generate a link that captures your current settings. Bookmark or save that link to return to the exact same view later.
Export the map. Click Export to download a high-resolution PNG of the current map view, complete with title, legend, and attribution.
Point query. With the point query tool active (on by default), click anywhere on the map to see the exact value and color at that location.
Data Archive. Click Data Archive in the toolbar to access the full historical archive of drought data for download and analysis.
Methods documentation. Click Methods to read detailed documentation on how each variable is calculated and what data sources are used.
Send feedback. Found a bug or have a feature request? Click Feedback to let us know.
Open source. Click GitHub to view the source code and data pipeline behind this dashboard.
